# CareerPilot AI

> AI-powered career assistant for finding flexible remote jobs, generating tailored resumes and cover letters, tracking applications, and preparing for interviews.

## Features

- Remote job collection
- Part-time and flexible schedule filtering
- Intelligent job matching
- Resume generation
- Cover letter generation
- Company research
- Application tracking
- Interview preparation
- Career analytics
